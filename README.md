# 2018年个人IPO体系
### IPO知识输出模型
> * ***I***：输入问题（Input a question），以持续解决问题为目标；
> * ***P***：解决问题（Problem solving），以整合所学知识为手段；
> * ***O***：输出产品（Output），通过咨询研发、授课整合和写作，让思想产品化；
